Koble til database

Appen bruker Supabase som gratis database. Dataene dine lagres sikkert online — tilgjengelig fra alle datamaskiner, aldri tapt.

1
Gå til supabase.com, opprett gratis konto og et nytt prosjekt. Velg Frankfurt som region.
2
Gå til SQL Editor → New Query, lim inn innholdet fra 1_supabase_setup.sql og klikk Run. Gjør det samme med 2_supabase_auth_setup.sql.
3
Gå til Authentication → Email Templates og aktiver «Confirm email». Gå deretter til Project Settings → API og kopier URL og anon key.
